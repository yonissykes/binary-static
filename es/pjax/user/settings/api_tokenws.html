Token API

Con acceder a nuestras aplicaciones móviles y otras aplicaciones de terceros, primero debe generar un token de la API.

  • Simplemente haga clic en "Crear" para generar el token; a continuación, cópielo y péguelo en la aplicación.
  • Elija el tipo específico de token de la API que necesita, basándose en las capacidades que desea hacer disponibles.
